#nukupuns 自动编曲程序
Nukupuns 短嘴导鹗雀，于1890年灭绝。这里是指音乐分析和自动作曲程序。<br />

## 特性
1、歌曲电子化，歌曲格式表示；<br />
2、歌曲分析，和弦分析和歌词分析；<br />
3、根据曲谱自动编配吉他和弦。<br />

## 更新
2020-04-01 初版 <br />
1.编写乐器表；<br />
2.编写全音阶、调性和拍子等音乐基础表；<br />
3.编写吉他和弦对照表和和弦指法表。<br />

## 设计理念
1、根据简谱和和弦走势、和弦衔接理论自动编配吉他谱；<br />
